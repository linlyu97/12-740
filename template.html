Introduction

  • This project is designed to produce a basic indoor air quality real-time monitoring system. The features designed to be measured through using Raspberry Pi are: smoke, humidity, and temperature, which are common criterions for quantifying the indoor air quality status. The outcome of the project will be real-time data collecting system and an alter system when any one of the quantities go over the theoritical healthy standard.
